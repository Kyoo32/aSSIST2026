## Context Layers And AI Memory

### Why Context Matters

- AI 시스템에서 컨텍스트는 단순 참고 정보가 아니라, 답변의 품질과 일관성을 좌우하는 핵심 요소다.
- 특히 에이전트형 AI에서는 현재 대화뿐 아니라 과거 지식과 관계 구조까지 함께 다뤄야 한다.

### Three Layers

- **상호작용 레이어**: 사용자와 AI가 만나는 접점이다.
  - 질문, 답변, 피드백이 오가는 층이다.
  - 실제 대화 흐름을 컨텍스트와 연결하는 역할을 한다.

- **컨텍스트 레이어**: 에이전트의 기억과 이해를 담당한다.
  - 상황, 데이터, 관계를 유지한다.
  - RAG, Vector DB, Knowledge Graph 같은 구성요소가 들어간다.
  - 단기 기억은 현재 대화, 장기 기억은 지식과 데이터에 가깝다.

- **모델 레이어**: 실제 추론과 생성을 수행한다.
  - 답변 생성
  - 의사결정
  - 추론 엔진 실행

### Building A Context Layer

- **구조 컨텍스트**: 단위 업무 분해, 도메인 전문가 검증, 역할-업무-역량 그래프 설계
- **행동 컨텍스트**: 수용, 판단, 리뷰 패턴, 지속 가능성
- **비교 컨텍스트**: 조직 간 벤치마크와 비교 프레임

> 핵심 포인트: 좋은 모델만으로는 충분하지 않고, 어떤 맥락을 유지하느냐가 실제 에이전트 성능을 좌우한다.
