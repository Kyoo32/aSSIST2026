## ARIMA Family And Diagnostics

### Core Models

| 모델 | 핵심 아이디어 |
| --- | --- |
| **AR** | 현재 값이 과거 값에 의존한다 |
| **MA** | 현재 값이 과거 오차에 의존한다 |
| **ARMA** | 과거 값과 과거 오차를 함께 고려한다 |
| **ARIMA** | 차분 후 AR과 MA를 결합한다 |
| **ARIMAX** | ARIMA에 외부 변수를 추가한다 |
| **SARIMA** | ARIMA에 계절성 구조를 추가한다 |

### How To Read Them

- **AR 모델**은 자기상관 구조를 직접 반영한다.
- **MA 모델**은 백색잡음의 누적 효과를 통해 현재를 설명한다.
- **ARMA 모델**은 정상 시계열에서 기본 조합으로 많이 쓰인다.
- **ARIMA 모델**은 비정상 시계열을 다루기 위한 대표 모형이다.
- **ARIMAX**는 외생 변수까지 반영하고 싶을 때 적합하다.
- **SARIMA**는 계절 패턴이 강한 데이터에서 자주 사용된다.

### Diagnostics And Tests

- **ADF 검정**: 단위근 존재 여부를 확인한다.
- **KPSS 검정**: 정상성을 다른 관점에서 점검한다.
- **Box-Jenkins Method**: 식별, 추정, 진단의 흐름으로 모델을 선택하고 검토하는 절차다.

### Volatility Modeling

- **ARCH**: 오늘의 변동성은 어제 충격의 크기에 의해 결정된다고 본다.
- **GARCH**: 오늘의 변동성은 어제 충격뿐 아니라 어제 변동성 수준 자체에도 영향을 받는다고 본다.

### Limitation Of ARIMA

- ARIMA는 강력하지만, 기본적으로 한 변수의 현재가 그 변수 자신의 과거에 크게 의존한다고 본다.
- 여러 변수의 상호작용이 중요하면 단변량 모델만으로는 부족할 수 있다.
